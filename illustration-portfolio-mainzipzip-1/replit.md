# Pranav Borse - Portfolio Website

## Overview
A modern, responsive portfolio website for Pranav Borse showcasing skills, experience, and projects as a Full Stack Developer.

## Tech Stack
- **Frontend**: React.js with Vite
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Icons**: React Icons

## Project Structure
```
├── public/
│   └── assets/           # Static assets (images, resume PDF)
├── src/
│   ├── components/       # React components
│   │   ├── About.jsx
│   │   ├── Contact.jsx
│   │   ├── Footer.jsx
│   │   ├── Navbar.jsx
│   │   ├── Projects.jsx
│   │   └── Skills.jsx
│   ├── pages/
│   │   └── Home.jsx      # Hero section
│   ├── utils/
│   │   └── CursorAnimation.jsx
│   ├── App.jsx           # Main app component
│   ├── main.jsx          # Entry point
│   └── index.css         # Global styles
├── index.html
├── vite.config.js
├── tailwind.config.js
└── package.json
```

## Features
- Responsive design for all screen sizes
- Smooth scroll navigation
- Animated sections using Framer Motion
- Custom cursor animation
- Mobile-friendly hamburger menu
- Downloadable resume
- Contact form
- Social media links (LinkedIn, GitHub, Email)
