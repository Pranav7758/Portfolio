# Pranav Borse - Portfolio

## Overview
A personal portfolio website for Pranav Borse showcasing skills, projects, education, certificates, and artwork. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Key Features
- Responsive design for all screen sizes
- Smooth animations with Framer Motion
- Custom cursor animation
- Type animation on hero section
- Art Gallery with click-to-view modal
- Resume download functionality
- Social links (Gmail, LinkedIn, GitHub)

## Tech Stack
- React 19
- Vite 6
- Tailwind CSS
- Framer Motion
- React Icons

## Running the Project
```bash
cd illustration-portfolio-mainzipzip-1
npm install
npm run dev
```
Server runs on port 5000.
